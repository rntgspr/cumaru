# Drafts

Working notes and guided rough thinking. Drafts may become memories, may remain
as notes, or may be discarded.

## Rules

- **One directory per draft.** Use `drafts/<slug>/index.md`.
- **Drafts are not memory.** They can be messy, partial, exploratory, or
  user-authored.
- **Promote only when useful.** A draft becomes memory only after distillation
  into a self-contained node under `memories/`.
- **Track lineage.** Use `derived-from` when a draft came from `inbox/` or
  another draft.

## When to use

- Freeform notes about a subject.
- LLM-guided thinking that is not ready to become permanent memory.
- Intermediate synthesis from multiple captures.

## When NOT to use

- Raw source material -> `inbox/`.
- Stable, loadable memory -> `memories/`.
